# CodeAlpha_HangmanGame

## Description
This is a simple text-based Hangman Game developed using Python. The player has to guess a hidden word one letter at a time.

## Features
- Word guessing game
- Letter-by-letter input
- Displays guessed letters
- Win condition when the word is completed
- Console-based interface

## Technologies Used
- Python

## How to Run
1. Open the project in Python or Pydroid 3.
2. Run the `hangman.py` file.
3. Enter one letter at a time to guess the word.
4. Complete the word to win the game.
